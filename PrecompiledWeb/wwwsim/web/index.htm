FUNDAMICRO, ofrece a todas las Instituciones de apoyo financiero a la Micro y Pequeña Empresa, el Sistema Informático para Microfinanzas (SIM), este es un software especializado para la Gestión y Administración de Carteras de Crédito. Está integrado por los siguientes Módulos:

1. Módulo de Administración
2. Módulo de Créditos
3. Módulos de Contabilidad
4. Módulos de Calificación de Cartera
5. Módulo de Ahorros
6. Módulo de Clientes
7. Módulos de Reportes.

Los ambientes de trabajo del SIM, han sido diseñados para operar en la plataforma Visual Fox Pro y Visual Basic .NET-SQL SERVER, dependiendo de las necesidades y presupuesto de las Instituciones. El SIM, es un Sistema de Información que responde a las necesidades de cualquier Institución financiera.
